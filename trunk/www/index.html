Project Description:

Rearranger is a plugin for IntelliJ IDEA which rearranges (reorders) Java source code.  With Rearranger you can:

  • Rearrange class and class member declarations according to a user-specified order and/or method call hierarchy;
  • generate section-marking comments;
  • control spacing within and between methods and classes; and
  • manually rearrange items.  

Comments, ratings, and downloads of current and previous versions of the plugin are available at JetBrains' IntelliJ IDEA Plugin site.
